Notes:

An animation changes an element from one style to another gradually.

To use an animation you must prepare a '@keyframe' of what you want it to do at the start, middle, and end. For example:

@keyframes example {
0% {background-color: transparent;}
100% {background-color: black}
}

You must also bind the keyframes to the block of code you want it to run on. For example:

.example {
width: 100px;
height: 100px;
animation-name: example;
animation-duration: 5s;
}

Now, our animation only happens one time on the page loading! There's a way that we can get it to load multiple times, or even infinite times. This is how:

.example {
width: 100px;
height: 100px;
animation-name: example;
animation-duration: 5s;
animation-iteration-count: (number or infinite)
}

Now the code animation will repeat forever!
