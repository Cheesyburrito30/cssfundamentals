Classes

A CSS Class is an identifier of an element which can be used multiple times on a page.

Memory device: Classes in school are periods. remember to use a . whenever you want to reference a class.

  • Classes start with . in the selector (.anything-you-want-to-name-it> )
  • Again, the same class name can be used more than wonce within the same file.
  • the clas keyword refers to the class name that you set in the HTML page in the specific tag
  • in the CSS file, you can identifiy a class on the whole, or you can identify particular parts to style (EX: ptags, h3, or li tags)
